# Input

Input is the component that you use, for get user taps and clicks.

```js
import { Input } from 'domeui-react'
```

### Classnames

```css
/* Container */
.DUI-containerInput {
}

/* Label */
.DUI-label {
}

/* Flex Input (Suport Icons and Input) */
.DUI-flexInput {
}

/* Icon Outside Left */
.DUI-iconOutsideLeft {
}

/* Icon Inside Right */
.DUI-iconInsideRight {
}

/* Input */
.DUI-containerInput {
}

/* Icon Inside Right */
.DUI-iconInsideRight {
}

/* Icon Outside Right */
.DUI-iconOutsideRight {
}
```
