# Button

Button is the component that you use, for get user taps and clicks.

```js
import { Button } from 'domeui-react'
```

### Classnames

```css
/* Button */
.DUI-button {
}

/* Icon Left */
.DUI-iconLeft {
}

/* Icon Center */
.DUI-iconCenter {
}

/* Icon Right */
.DUI-iconRight {
}
```
